Streamlit 소개와 설치

2023-01-24

Streamlit 소개

Streamlit은 머신 러닝 및 데이터 과학 프로젝트를 위한 오픈 소스 앱 프레임워크이다. 실험 결과를 간단히 프로토타이핑 하여 공유할 때 유용하게 사용할 수 있다.

Streamlit 설치

$ pip install streamlit

streamlit의 hello 앱을 실행한다.

$ streamlit hello

사용자 앱 실행

streamlit에서 제공하는 API를 이용해 출력에 사용할 구성을 적용한다.

[app.py]

import streamlit as st

st.title ("app title입니다.")
st.header("header입니다.")
st.markdown("markdown입니다.")
st.subheader("subheader입니다.")
st.caption("caption입니다.")
st.code("x=2023")
st.latex(r''' a+a x^1+a x^2+a x^3 ''')

streamlit을 이용해 app.py을 실행한다.

$ streamlit run app.py

브라우저(http://localhost:8501/)에 앱 실행결과가 출력된다.
