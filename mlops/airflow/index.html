Airflow 설치

2023-01-24

apache/airflow 도커 이미지는 다음과 같은 사항이 포함되어 있다.

  • Airflow 설치시 필요한 패키지가 포함된 기본 OS(안정적인 Debian OS)
  • 릴리스된 Airflow의 MINOR 버전에 대해 릴리스 당시 지원되는 버전의 기본 Python 설치
  • 지원되는 데이터베이스에 연결하는 데 필요한 라이브러리

Airflow 설치를 간소화 하기 위해 bitnami/airflow 이미지를 이용한다.

$ docker pull bitnami/airflow

docker compose yml 파일을 다운로드 받는다.

$ curl -sSL https://raw.githubusercontent.com/bitnami/containers/main/bitnami/airflow/docker-compose.yml > docker-compose.yml
$ docker-compose up -d

도커 컴포즈로 컨테이너를 실행하면 관련 컨테이너가 함께 실행된다.

기본적으로 아래와 같이 db가 초기화 되어 있고 8080 웹서버가 실행되어 있다.

airflow db init
airflow webserver -p 8080

도커 컴포즈로 Airflow 관련 도커 컨테이너가 실행되면 아래 URL로 Airflow 로그인 페이지 접속이 가능하다.

http://127.0.0.1:8080/

Airflow를 접속하기 위한 초기의 id와 password는 아래와 같다.

id : user
password : bitnami
