2020-02-10

SQuAD

최근에는 기계 독해를 평가하기 위한 SQuAD(The Stanford Question Answering Dataset)와 같은 위키피디아 기반 데이터 셋이 있습니다. 이 데이터셋은 기계 독해 알고리즘을 객관적으로 평가할 수 있는 데이터 셋으로 알고리즘의 우수성을 평가하기 위해 리더 보드를 운영하고 있습니다. 전체 데이터셋의 사이이즈는 학습셋은 40MB, 개발 셋은 4MB정도에 불과합니다.

  • https://rajpurkar.github.io/SQuAD-explorer/

SQuAD 2.0 데이터 셋의 구조는 다음과 같습니다.

{
  "version": "v2.0",
  "data": [
    {
      "title": "Normans",
      "paragraphs": [
        {
          "qas": [
            {
              "question": "In what country is Normandy located?",
              "id": "56ddde6b9a695914005b9628",
              "answers": [
                {
                  "text": "France",
                  "answer_start": 159
                },
                {
                  "text": "France",
                  "answer_start": 159
                },
                {
                  "text": "France",
                  "answer_start": 159
                },
                {
                  "text": "France",
                  "answer_start": 159
                }
              ],
              "is_impossible": false
            }
          ]
        }
      ]
    }
  ]
}

KoQuAD

KoQuAD(The Korean Question Answering Dataset)는 LG CNS가 공개한 Machine Reading Comprehension 데이터셋 입니다. 현재는 koQuAD 2.0 버전이 공개 되었습니다. koQuAD 2.0은 질문답변 2만 쌍을 포함한 총 10만쌍 이상 규모입니다. 이들 질문답변 쌍을 수집하기 위해 한국어 위키피디아 페이지 뷰 15만건과 임로 선된 5만의 문서를 수집이 되었습니다. koQuAD의 질문은 텍스트
