About

2020-01-15

사용자에게 행복을 주는 서비스를 만드는 해피그래머의 개발 블로그입니다. 저의 관심 영역은 AI, NLP, AI Service 입니다. 이 블로그는 AI 서비스를 만드는데 필요한 기반 기술들을 소개하고 다루는 것에 초점이 있습니다. 다음은 제가 좋아하는 영국 속담입니다.

A smooth sea never made a skilled mariner.

이 블로그의 주제

  • IT와 삶의 지혜 : IT 관점에서 삶의 지혜가 무엇일지에 대한 주제를 다룹니다.
  • 프로그래밍 가이드 : 일반적인 프로그래밍과 관련한 주제를 다룹니다.
  • 자연어처리 : 언어 처리와 관련한 전통적인 기술 부터 최신의 기술을 포함해 다룹니다.
  • 한글 공학 : 한글의 역사, 한글 문법 등을 소개하고 한글을 컴퓨터와 접목할 수 있는 방안등을 소개합니다.
  • 텐서플로우 : 머신러닝 모델 알고리즘 구축의 연구 개발을 지원하고 있는 텐서 플로우를 다룹니다.
  • 논문 리뷰 : 언어처리, 머신 러닝 논문 내용을 도해 등으로 알기 쉽게 소개 하기 위한 목적입니다.
  • 스프링 프레임워크 : 대규모의 엔터프라이즈급 서버 구축에 활용되는 스프링 프레임워크를 소개합니다.
  • 파이썬 : 데이터 분석, 데이터 모델링, 알고리즘 설계 등에 효율성 있는 파이썬을 소개합니다.
  • Angular : 클라이언트 인기 프레임워크인 Angular를 소개합니다.
  • 도커 : 컨테이너 기반 서비스에 대한 내용을 다룹니다.
  • 우주 : 우주에 일어난 신비한 일들에 대해 소개합니다.
