NLP 데이터셋 소개

2020-02-10

SQuAD

최근에는 기계 독해를 평가하기 위한 SQuAD(The Stanford Question Answering Dataset)와 같은 위키피디아 기반 데이터 셋이 있습니다. 이 데이터셋은 기계 독해 알고리즘을 객관적으로 평가할 수 있는 데이터 셋으로 알고리즘의 우수성을 평가하기 위해 해리 보드를 운영하고 있습니다. 전체 데이터셋의 사이이즈는 학습셋은 40MB, 개발 셋은 4MB정도에 불과합니다.

  • https://rajpurkar.github.io/SQuAD-explorer/

SQuAD 2.0 데이터 셋의 구조는 다음과 같습니다.

{
  "version": "v2.0",
  "data": [
    {
      "title": "Normans",
      "paragraphs": [
        {
          "qas": [
            {
              "question": "In what country is Normandy located?",
              "id": "56ddde6b9a695914005b9628",
              "answers": [
                {
                  "text": "France",
                  "answer_start": 159
                },
                {
                  "text": "France",
                  "answer_start": 159
                },
                {
                  "text": "France",
                  "answer_start": 159
                },
                {
                  "text": "France",
                  "answer_start": 159
                }
              ],
              "is_impossible": false
            }
          ]
        }
      ]
    }
  ]
}
