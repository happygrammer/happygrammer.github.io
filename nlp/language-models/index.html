언어 모델

2020-01-30

언어 모델(language model)은 단어 시퀀스(word sequence)를 이용한 확률 모델입니다. 언어 모델은 기계 번역, 음성 인식, 오타 교정(spell correction), 손글씨 인식(handwriting recognition), 문서 요약, 질의 응답 등과 같이 최종 출력인 단어 시퀀스를 예측하거 예측해야 하는 태스크에 사용됩니다.

어어 모델에서 고려하는 단어 시퀀스는 n개의 단어가 연속해서 나는 확률 P(w1,w2,w3,…,wn)와 같습니다.

P(W)=P(w1,w2,w3,...,wn)

즉, n개의 단어를 하나의 시퀀스로 보고 단어 시퀀스들의 확률 값들을 계산하면 언어 모델이 됩니다. 하나의 단어(w) 앞에 나온 n개의 워드 시퀀스(word sequence)을 고려해 다음에 나올 단어를 예측하는 확률 모델입니다.
