MLOps 플랫폼이란?

MLOps 플랫폼 기술은 상용을 위한 기계학습 애플리케이션 배포 및 확장과 관련한 기술이다. 크게 세가지 관점의 기술 내용을 포함한다.

  • ML 모델 상태 관리
  • E2E 머신러닝 오케스트레이션
  • 컨테이너 기술

ML 모델 상태 관리 기술은 학습 모델의 자동 배포와 학습된 모델의 생명 주기를 관리 하는 기술이다. 모델의 생성 날짜, 생성 주기, 모델의 성능 수치와 같은 모델의 특성 정보를 포함할 수 있다. 이 기술은 모델을 모니터링하여 모델의 사용량과 모델의 처리량을 지속적으로 모니터링하고 제어하는 것 뿐 아니라 신규 모델을 찾아 해당 모델의 사양을 확인하고 비교할 수 있는 기술을 포함한다. E2E 머신러닝 오케스트레이션 기술은 AutoML 기능을 활용해 AI 모델 생성의 복잡성을 낮추는 기술이다. 사용자가 머신러닝 지식 배경이 없더라도 AutoML의 기능을 활용하면 자동화된 모델을 생성할 수 있게 된다. 컨테이너 기술은 MLOps의 중요 과정들을 컨테이너하고 솔루션화 하여 MLOps 적용과 운영 과정의 복잡성을 줄이는 기술이다.

MLOps 최적화 기술

MLOps 플랫폼 기술은 AI 모델 생명주기를 보다 효율적이고 유연하게 관리하기 위해 최적화 기술 지원이 필요하다.

  • AutoML (Automated Machine Learning)
  • 파인튜닝 기술
  • GPU 최적화 기술

먼저 AutoML기술이다. AutoML기술은 데이터 전처리, 학습 과정의 매개변수인 하이퍼파라메터 탐색, 알고리즘 선택문제를 자동화하는 기술이다. 이 기술을 보다 솔루션할 수 있다면 사용자가 ML지식이 없더라도 비즈니스 환경에 맞는 최적의 모델을 생성하는데 도움이 될 수 있다.파인튜닝 기술은 사전 학습된 언어 모델(pretrained language model)을 추가로 재학습해 활용하는 기술을 의미한다. 모델은 정책에 따라 공개될 수 있으며, 비즈니스 또는 사용자의 요구에 따라 확장이 필요할 수 있다. 파인튜닝 기술을 활용하면 언어 모델의 확장하고, 모델 운영 비용을 대폭 절감하는데 도움이 될 수 있다. GPU 최적화 기술 은 모델 생성에 필요한 GPU 사용량을 효과적으로 관리하여 모델 생성과 운영 필요한 비용을 줄이는 기술이다. 하나의 AI 모델을 생성하기 위해서는 많은 GPU 자원이 필요하며 사용자에게는 부담이 된다. MLOps 관련 솔루션은 GPU 최적화 기술을 포함하여 적은 GPU 사용량으로도 동등 수준의 AI 모델 생성을 지원할 수 있어야한다. GPU 최적화 기술의 예로 NVIDIA/DALI, NVIDIA/TensorRT 등과 같은 기술이 있다.
