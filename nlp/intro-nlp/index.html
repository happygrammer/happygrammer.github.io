자연어 처리란 무엇인가?

2020-01-29

자연어 처리는 사람의 언어를 이해하고 컴퓨터와 사람이 상호 소통할 수 있게 하는 기술입니다. 자연어 처리의 궁극적인 목표는 컴퓨터가 사람과 같은 언어지능(linguistic Intelligence)을 갖추도록 하는 것 입니다. 언어지능은 말과 글을 이해하고 다시 활용할 수 있는 능력입니다. 컴퓨터가 말과 글의 의미와 의도를 이해하면 사람 처럼 주어진 문제를 해결 하거나 언어에 나타난 말들을 개념화하는 추상화 추론(abstract reasoning)도 가능하게 됩니다.

자연어 처리 과정

자연어 처리의 대상은 텍스트이며, 텍스트 중에서도 사람의 언어가 대상이 됩니다. 자연어 처리 과정은 다음과 같습니다.

형태소 분석 → 구문 분석 → 의미 분석 → 화용 분석

형태소 분석(morphological analysis)은 자연어의 최소 의미인 형태소를 식별하는 언어 처리 과정입니다. 구문 분석(syntax analysis)은 주어, 동사, 목적어등의 문장성분을 판별해 문장성분에 따른 문장 구조를 분석하는 과정입니다. 문장성분을 분석하기 위해 구문 분석기(parser)를 이용해 구문 분석을 수행합니다. 구문 분석 후 나타난 구조 애매성등의 해결이 필요하며, 구문 분석을 마치면 의미 분석(semantic analysis)단계를 진행합니다. 의미 분석은 형태소의 실재 의미가 무엇인지를 판별하는 과정입니다. 이후 화용 분석 단계로 대명사가 지시하는 것이 무엇인지(anaphora), 화자가 원하는 행위(speech act)가 무엇인지를 분석 하는 과정입니다.
