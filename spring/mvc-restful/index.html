@ControllerмҷҖ @RestController м–ҙл…ён…Ңмқҙм…ҳ

2019-11-20

@Controller м–ҙл…ён…Ңмқҙм…ҳмқ„ мқҙмҡ©н•ң MVC нҢЁн„ҙ

MVCлҠ” @Controllerм—җ кё°л°ҳн•ҙ мһ‘лҸҷн•ҳлҠ” View кё°л°ҳмқҳ кё°мҲ мқҙл©°, RESTлҠ” @RestController м–ҙл…ён…Ңмқҙм…ҳмқ„ мқҙмҡ©н•ҙ к°қмІҙ лҚ°мқҙн„° мқ‘лӢө кё°мҲ мһ…лӢҲлӢӨ. м•„лһҳ к·ёлҰјмқҖ м „нҶөм Ғмқё MVC нҢЁн„ҙ л°©мӢқмқҳ мІҳлҰ¬ кіјм •мқ„ ліҙм—¬мЈјкі  мһҲмҠөлӢҲлӢӨ.

мҠӨн”„л§ҒмқҖ мӮ¬мҡ©мһҗ мҡ”мІӯм—җ лҢҖн•ҙ Dispatcher ServletлҠ” мөңмў… мІҳлҰ¬ кІ°кіјлЎң ViewлҘј Returnн•ҙмӨҚлӢҲлӢӨ. DispatchServletмқҖ м•„лһҳ к°Ғ н•ҳмң„ мҡ”мҶҢл“Өмқ„ нҳём¶ңн•ЁмңјлЎңм„ң нҒҙлқјмқҙм–ёнҠём—җ мөңмў… http мқ‘лӢөмқ„ л§Ңл“ӯлӢҲлӢӨ. URIк°Җ мһ…л ҘлҗҳлҠ” Handler mappingмқ„ мқҙмҡ©н•ҙ м»ЁнҠёлЎӨлҹ¬лЎң м—°кІ°н•ҙ мӨҚлӢҲлӢӨ. м»ЁнҠёлЎӨлҹ¬м—җм„ңлҠ” MovelAndView к°қмІҙлҘј View ResolverлЎң л°ҳнҷҳн•©лӢҲлӢӨ. View ResolverлҠ” ModelAndView к°қмІҙлҘј мқҙмҡ©н•ҙ ViewлҘј мғқм„ұн•©лӢҲлӢӨ. view мғқм„ұм—җ мҙҲм җмқ„ л§һм¶”кі  мһҲмңјлҜҖлЎң mime typeм—җ л”°лқј мқ‘лӢөмқ„ кІ°м •н•  мҲҳ м—ҶмҠөлӢҲлӢӨ.

mim typeм—җ л”°лқј мқ‘лӢөмқ„ л§Ңл“ңлҠ” @ResponseBody м–ҙл…ён…Ңмқҙм…ҳ

@ResponseBody м–ҙл…ён…Ңмқҙм…ҳмқҖ мқ‘лӢө к°’мқ„ мЈјм–ҙм§„ mime type нғҖмһ…м—җ л”°лқј HTTP мқ‘лӢөн•©лӢҲлӢӨ. лӮҙл¶Җм ҒмңјлЎңлҠ” HttpMessageConvertersк°Җ мқ‘лӢө к°’мқ„ mime typeм—җ л”°лқј мөңмў… мқ‘лӢө к°’мқ„ л§Ңл“ӯлӢҲлӢӨ.

[к·ёлҰј] http мқ‘лӢөмӢң HttpMessageConvertersлҘј нҶөн•ң мқ‘лӢө

мҪ”л“ңлЎң ліҙл©ҙ @RequestMappingл§ҲлӢӨ н•ҳлӮҳмқҳ @ResponseBodyлҘј м„ м–ён•©лӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ member/hello.xmlлқјлҠ” мҡ”мІӯм—җ лҢҖн•ң xml мқ‘лӢөмқҖ м•„лһҳмҷҖ к°ҷмқҙ л§Ңл“ӯлӢҲлӢӨ.

@RestController м–ҙл…ён…Ңмқҙм…ҳмқ„ мқҙмҡ©н•ң REST мқ‘лӢө

@RestController м–ҙл…ён…Ңмқҙм…ҳмқҖ мҠӨн”„л§Ғ 4м—җм„ң мҶҢк°ңлҗң м–ҙл…ён…Ңмқҙм…ҳмһ…лӢҲлӢӨ. @ControllerмҷҖ @ResponseBody м–ҙл…ён…Ңмқҙм…ҳмқ„ н•©м№ң м—ӯн• мқ„ н•©лӢҲлӢӨ.

@RestController = @Controller + @ResponseBody

@ResponseBodyк°Җ мһҲм–ҙ mime typeм—җ л”°лқј @Controller мІҳлҹј viewлҘј л§Ңл“Ө мҲҳ лҸ„ мһҲкі , json к°қмІҙ м“°кё° мқ‘лӢөмқ„ н•  мҲҳ мһҲмҠөлӢҲлӢӨ. л””нҸҙнҠё м–ҙл…ён…Ңмқҙм…ҳмқҙлҜҖлЎң лі„лҸ„ м–ҙл…ён…Ңмқҙм…ҳмңјлЎң м„ м–ён•ҳм§Җ м•Ҡм•„лҸ„ лҗ©лӢҲлӢӨ. к·ёлҹ°лҚ° @RestControllerмқҳ лӘ©м ҒмғҒ view мқ‘лӢө ліҙлӢӨлҠ” object/pojo кё°л°ҳ мқ‘лӢөмқ„ мЈјлӘ©м ҒмңјлЎң н•©лӢҲлӢӨ.
