Git을 이용한 `버전 관리`는 S/W 개발 프로세스 중 하나가 되었습니다. 버전 관리를 이용하면 소스 코드 이력 관리를 통해 안정적인 소스 코드 관리 운용이 가능하지만  능숙한 사용까지 시간이 걸립니다. Git 사용을 돕기 위해 주요 Git 명렁어에 대한 다이어그램을 작성해 보았습니다. 보완이 필요한 점은 피드백 환영 합니다. 감사합니다.
