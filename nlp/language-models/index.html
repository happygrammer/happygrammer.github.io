[언어 모델] n-gram 모델, Bag-Of-Words 모델

2020-01-31

언어 모델(language model)은 단어 시퀀스(word sequence)를 전제로한 확률 모델로 단어A와 단어B가 연속해 출현하는 확률 분포를 고려하는 모델입니다. 다국어 번역, 음성 인식, 문서 요약 등과 같이 단어열을 출력해야 하는 태스크에 사용됩니다.

n-gram 모델

언어 모델의 가까운 예로 n-gram 모델이 있습니다. n-gram 모델은 하나의 단어(w) 앞에 나온 n개의 단어열(word sequence)을 고려하는 확률 모델입니다.

P(W)=P(w1,w2,w3,...,wn)

Bag-Of-Words(BoW)

Bag-Of-Words(BoW) 모델은 언어 모델에서 시퀀스를 고려하지 않는 언어 모델로 일찍히 컴퓨터 비전(computer vision) 분야에 사용되오던 모델입니다. Bag-Of-Words는 시퀀스 고려 없이 각 단어의 빈도수를 고려하는 모델입니다. BoW는 정보 검색에서 문서간의 유사도 계산에 사용되었습니다. BoW의 예로 문서에 ‘셔츠'가 2번, ‘바지'가 3번, ‘내복'이 5번 등장했다면 BoW로 [2, 3, 5]라는 벡터로 나타냅니다.

Bag-of-Words은 다음과 같은 제한 사항이 있습니다.

  • 맥락 의미 미반영
  • 어휘 벡터 사이즈 제한

맥락 의미 미반영 문제는 단어 시퀀스를 고려하지 않아 문맥에 따라 어떤 의미로 사용될지를 예측할 수 없는 문제입니다. 어휘 벡터 사이즈 제한 문제는 큰 문서의 단어를 벡터로 변화할 때 단어를 벡터로 변환을 무한하지 않다는 점입니다. 단어를 벡터로 변환할때 메모리 비용이 있고, 어휘 벡터 사이즈가 커질 수 록 계산 비용이 발생합니다.
