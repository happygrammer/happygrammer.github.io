2020-12-31

데이터의 종류

머신 러닝에서 문제 유형에 따라 사용하는 데이터가 다르다. 데이터는 크게 형태에 따라 음성, 이미지, 텍스트로 나뉜다. 이중 텍스트 데이터의 예로 위키피디아 데이터가 있다.

데이터 레이블링

데이터 양이 많더라도 데이터 레이블링이 없다면 모델 학습이 잘 진행 되지 않을 수 있다. 학습이 잘 되지 않는다면 데이터 레이블링을 통해 해결이 필요하다.

  • 데이터 학습 편중 : 데이터 레이블링이 잘 되어 있더라도 특정 분류의 데이터가 상대적으로 데이터 레이블링이 되어 있지 않다면 학습시 사용하는 loss function에 의해 데이터 레이블링이 많이 되어 있는 분류로 편중되어 학습될 수 있다. 이러한 경우 과소 표집 방식으로 데이터 레이블링이 많은 데이터의 레이블을 누락 시켜 데이터 균형을 맞출 수 있다.
  • 적은 학습 데이터 : 만약 데이터 균형이 맞지만 데이터가 적다면 원샷러닝(one-shot learning) 또는 퓨샷 러닝(few-shot learning)을 이용해 적은 데이터에서도 학습이 가능한 방식으로 학습 알고리즘을 변경할 수 있다.

데이터 표준화

데이터 레이블링을 보다 원할히 수행하려면 수치, 카테고리와 같은 데이터를 표준화할 필요가 있다. 예를 들어 150cm 와 1.5미터는 150cm로 표준화한다. 예를 들어 성별을 카테고리 하면 0,2로 구분할 수 있다.
