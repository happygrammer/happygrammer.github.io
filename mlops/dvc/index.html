DVC 소개와 설치

2023-01-24

데이터 버전 관리

대용량 파일 버저닝을 위한 도구로 git-lfs, LakeFS, DVC, DeltaLake, dolthhub, Pachyderm, Neptune 등이 있다. 이중 DVC에 대해 알아 보도록 한다.

DVC 소개

데이터 버전 제어 (DVC)는 데이터 및 모델 버전을 캡쳐할 수 있는 오픈소스 기반의 무료 CLI 도구이다. DVC를 사용 하여 데이터의 스냅샷 을 만들고, 이전 버전을 복원 하고, 실험을 재현할 수 있다. 용량 제한이 있는 git과 달리 git에 DVC CLI를 추가로 이용하면 대용량 파일, 데이터셋 디렉토리, 머신러닝 모델 등을 작은 메타 파일로 대체하여 데이터 버전 관리를 지원한다.

DVC 설치

DVC(Data Version Control) 설치를 진행한다. MacOS를 이용하고 있다면 brew를 이용해 설치한다.

$ brew install dvc

DVC 저장소 초기화

설치 후 dvc-test-repo 경로를 만들고 dvc 저장소를 초기화한다.

$ mkdir dvc-test-repo
$ cd dvc-test-repo
$ dvc init
Initialized DVC repository.

You can now commit the changes to git.

DVC commit

첫번째 commit을 수행한다.

$ git commit -m "Initialize DVC"
[master (최상위-커밋) d2165ca] Initialize DVC
 4 files changed, 9 insertions(+)
 create mode 100644 .dvc/.gitignore
 create mode 100644 .dvc/config
 create mode 100644 .dvcignore
 create mode 100644 test.txt

이후 원격 저장소(Amazon S3, SSH, Google Drive, Azure Blob Storage, and HDFS) 주소를 추가하고 add 할 수 있다. 아래 예시는 Amazon S3를 기준으로 한다.

dvc remote add -d storage s3://mystorage/dvcstore
git add .dvc/config
git commit -m "add dvc config"

원격 저장소에서 data를 가져오려면 pull 명령을 사용한다.

dvc pull

DVC add/push

dvc 명령은 dvc add 명령과 dvc push 명령을 주로 함께 사용한다.

$ dvc add ./data/test.txt
$ dvc push

dvc add 명령을 이용해 스테이지에 추가한다.

$ dvc add ./data/test.txt
100% Adding...|█████████████████████████████████████████████████████████████████████████████████████████████████████████████████|1/1 [00:00,  5.07file/s]
                                                                                                                                                         
To track the changes with git, run:

	git add data/.gitignore data/test.txt.dvc

To enable auto staging, run:

	dvc config core.autostage true

다음과 같은 ./data/test.txt.dvc 파일이 생긴다.

outs:
- md5: 51b98813c49f7031395c63a33f7db50b
  size: 9
  path: test.txt

마지막으로 원격 저장소에 push한다.

dvc push

dvc는 버전 제어 시스템이 아니다. Git 자체가 버전 컨트롤을 담당한다.

DVC 버전이동

dvc 버전 이동을 하려면 git 명령어와 함께 사용한다.

git checkout HEAD~1
dvc checkout
